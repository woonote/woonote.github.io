In order for Woonote to work, we need to briefly explain some basics here. As a reminder, pay attention to the distinction between user identities.

Webmaster: Have administrator rights, you can install plugins.

User: Users with accounts on WordPress sites, author or above, otherwise you will not have permission to upload images.

1.Download

# Download the note software corresponding to your system.

# (Only for Webmaster)Download the WordPress plugin. Because it depends on the JWT Auth plugin, you also need to download it together.

Extension link: JWT Auth

2.Install

# Install the note software.

# (Only for Webmaster)Install the WordPress plugin(Includes JWT Auth plugin), For details:
https://wordpress.org/documentation/article/manage-plugins/#upload-via-wordpress-admin

3.Configurate

# Run note software, Menu>Settings to open the configuration pop-up box, fill in the URL link of your website.

# (Only for Webmaster)Login to the website backend, switch to Settings>WooNote page, fill in your AppId and AppKey:

4.Activate

# After the user login, Menu>Settings to open the configuration pop-up box, click the "Activate" button to activate the account and use it.(Only activated when user login for the first time)
